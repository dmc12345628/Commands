Angular
=======

### Installez le compilateur TypeScript
```
npm install typescript -g
```

### Compilez le fichier source TypeScript
```
tsc [fichier].ts
```

### Installer angular client
```
npm install -g @angular/cli
```

### Créer un projet angular cli
```
ng new my-dream-app
```

### Démarrer le serveur
```
ng serveur
```
